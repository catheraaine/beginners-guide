JavaScript

Summary

Intro

JavaScript is a scripting language on the client side of web pages that allows developers to create dynamically generate content through object oriented or functional programming patterns.

Run JS

Reference an external script file by adding a script tag at the end of your html document. Then javascript will run from top to bottom of your file. Use ES6 imports to separate different aspects of your program in different files. This will make it easier to catch bugs and stay organized. Most articles online do not use ES6, yet that is the standard now.

JavaScript Projects

Start A Project

Escape tutorial purgatory by using a project to learn JS. Pick something you are interested in, whether accessibility, gaming, storytelling, data. Take a problem you would like solved related to your interest and use Javascript to solve it. Start off with a discovery session by doing some research and brainstorming before you write any code. Then get your first draft of a program up and running. Start refactoring your code in small sections after you have it working. Make sure to keep everything in a GitHub repo.

Complete Katas

Complete at least two katas. First get them working just with node. Then bring them to the browser, and give them some styling. Create a GitHub repo before you start, and push all of your changes as you go to the repo.
